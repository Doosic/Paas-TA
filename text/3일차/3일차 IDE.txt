
	* IDE
		프로그램 개발에 관련된 모든 작업을 하나의 프로그램 안에서
		처리하는 환경을 제공하는 소프트웨어
		VS, ECLIPSE 등....
		
		- 클라우드 IDE
		  로컬에서 사용가능한 기존 IDE의 불편함을 줄이기 위해 등장
		  기존것을 대체하기보다는 불편함을 감소해보고자 등장
		  
		- 장점
		  물리적인 컴퓨터에 종속되지않음
		  여러 사람이 동시에 작성 가능해 페어 프로그래밍을 하기 편함
		  접근 권한을 중앙에서 관리해 보안성을 높일수 있다
		  수정 기록이 웹에 실시간으로 남기 때문에 향후 코드 내용 추적 용이
		
		- 단점
		  기존 설치형 IDE보다 성능이 낮고 기능이 부족
		  
		
		- 핵심정리
		  Eclipse 에서 cloudfoundry plugin을 통하여 api에 접속할 수 있다.
		  프로젝트를 배포할 수 있다
		  접속한 api 데시보드에서 app을 관리할 수 있다.
		  서비스를 생성하여 앱과 연동할 수 있다.
